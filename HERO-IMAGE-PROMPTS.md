# Hero Image Generation Prompts

## Project Context Descriptions

### COPPA - Process & Technology
**What it does**: Automated tool to validate if all permitted API calls in Kids content applications are approved by the legal team, ensuring federal COPPA (Children's Online Privacy Protection Act) compliance. Previously required hours/days of manual work; now fully automated in minutes.

### Onboarding Automation Setup  
**What it does**: Video training library created by team members showing how to set up different platforms for automated testing. Covers device preparation, proxy configuration, and framework setup for each platform (WCTV, Web, Roku, Apple TV, iOS, Android, Fire TV). Enables new team members to independently configure platforms in hours instead of days/weeks without expert help.

### Test Manager Bot & EVA
**What it does**: Company's first AI-powered chatbot that unifies knowledge from scattered internal systems (Confluence, JIRA, GitHub, TestRail). Uses RAG (Retrieval Augmented Generation) to help QA engineers create tests, detect duplicates, and clarify requirements. Also assists Product Owners and Developers with documentation access and technical Q&A.

### Data Observability
**What it does**: Transforms traditional QA from reactive to proactive using real-time monitoring. Instead of running 500-600 redundant tests per platform (70% repeated steps), implements assertion points in production logs that alert when issues occur. Eliminates device cloud costs, catches bugs before users see them, and reduces testing from days to hours with minimal maintenance. Uses Monte Carlo as AI analyzer for monitoring.

## General Design Specifications for All Images:

### Background:
- Dark navy gradient (#0A1628 to #0D1F35 or #1e293b to #0f172a)
- Subtle grid pattern overlay (very faint, 2-5% opacity)
- Professional tech aesthetic

### Typography:
- Modern sans-serif (Inter, SF Pro, System UI, Arial)
- Section headers: Bold, 20-24px
- Metrics/numbers: Bold, 48-72px
- Body text: Regular, 12-14px
- All text crisp and readable

### Visual Elements:
- Rounded corners on cards/boxes (8-16px radius)
- Subtle shadows for depth
- Glow effects on key metrics (use filter/blur)
- Flowing connection lines (curved paths)
- Animated particles along data streams (small circles)
- NO photographs - purely iconographic and data visualization

### Color Palette:
- **Success/Positive**: Green (#10b981, #22c55e)
- **Warning/Before**: Red/Orange (#ef4444, #f59e0b)
- **Info/Process**: Blue (#3b82f6, #0066cc)
- **AI/Intelligence**: Purple (#8b5cf6)
- **Data**: Cyan (#00d9e0, #06b6d4)
- **Text Primary**: White (#ffffff)
- **Text Secondary**: Light gray (#cbd5e1, #94a3b8)
- **Backgrounds**: Navy (#1e293b, #0f172a)

### Export Settings:
- Format: PNG-24 or WebP
- Resolution: 1920x1080px
- Optimization: Compress to under 500KB (under 300KB preferred)
- Color profile: sRGB
- Ensure all text is readable when scaled to 800px width

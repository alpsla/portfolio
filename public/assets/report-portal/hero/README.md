# Report Portal Hero Image

This directory contains prompts and assets for the Report Portal project hero image.

## Files

- 'SHORT_PROMPT.txt' - Quick prompt for AI image generation
- 'image_prompt.txt' - Detailed prompt with full specifications
- 'README.md' - This file
- 'report_portal_hero.svg' - The generated hero image

## Usage

Use the prompts in 'SHORT_PROMPT.txt' or 'image_prompt.txt' with an AI image generator (DALL-E, Midjourney, Claude, etc.) or design tool to create the hero image.

## Specifications

- **Dimensions**: 1920x1080px (16:9 aspect ratio)
- **Format**: SVG preferred, or PNG/WebP
- **File size**: Under 500KB optimized
- **Style**: Modern TestOps/analytics dashboard visualization
- **Theme**: Centralized test reporting with data flow visualization

## Design Elements

The hero image should show:
1. Multiple test frameworks on the left
2. Report Portal platform in the center
3. Two dashboard outputs on the right (Test Results & Analytics)
4. Flowing data streams connecting elements
5. Metrics highlighting key capabilities
